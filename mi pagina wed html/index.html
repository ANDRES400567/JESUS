Selectores CSS

Un selector de CSS seleccione los elementos HTML a las que desean aplicar estilo.

Los selectores se utilizan para "buscar" (o seleccionar)los elementos HTML a los que desea aplicar estilo

Podemos dividir los selectores css en cinco categorias:

  1. Selector universal
  2. Selectores por etiqueta
  3. Selectores por clase
  4. Selectores por id

1.El selector universal de css

El selector universal (*) selecciona todos los elementos HTML de la pagina.

Ejemplo:La siguiente regla css afectara a los elementos HTML de la pagina


*{
texto-align: centrer;
color: red;
}
